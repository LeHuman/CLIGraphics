Mouse class
#include <mouse.hpp>

C++ singleton class wrapper around the manymouse library. Offers manual polling or automatic polling via a std::jthread.

Note

Multiple mice not supported, only the first mouse found is used.

Function documentation

int Mouse::setModulus(int xMod, int yMod)

Set a modulus value for mouse coordinates using a single magnitude for either coordinate direction.

Parameters
xMod The modulus value for Mouse::x in either direction.
yMod The modulus value for Mouse::y in either direction.
Returns
0 if modulus was set
1 if modulus was not set

Note

Overrides Mouse::setClamp until disabled

Set all to 0 to disable.

int Mouse::setClamp(int xMax, int yMax)

Set a clamping value for mouse coordinates using a single magnitude for either coordinate direction.

Parameters
xMax The max value Mouse::x should be in either direction.
yMax The max value Mouse::y should be in either direction.
Returns
0 if clamp was set
1 if clamp was not set

Note

Set all to 0 to disable clamping.

Recommended to set before calling Mouse::startPolling.

int Mouse::setClamp(int xMax, int yMax, int xMin, int yMin)

Set a clamping value for mouse coordinates.

Parameters
xMax The max value Mouse::x should be.
yMax The max value Mouse::y should be.
xMin The min value Mouse::x should be.
yMin The min value Mouse::y should be.
Returns
0 if clamp was set
1 if clamp was not set

Note

Set all to 0 to disable clamping.

Recommended to set before calling Mouse::startPolling.

int Mouse::startPolling(int updateTime_ms = 20)

Parameters
updateTime_ms The time interval in milliseconds at which the polling loop will be executed to check for mouse input.
Returns
0 if the polling thread started successfully.
-1 if the polling thread was already started.
1 if there were no mice detected (polling thread killed).
2 if the manymouse library failed to initalize (polling thread killed).

Initiates a polling loop with an update time in milliseconds.

void Mouse::stopPolling(bool join = true)

Parameters
join Whether the calling thread will wait for the polling thread to finish. Default is true.

Stops the polling thread and optionally waits for the polling thread to finish.

Variable documentation

const volatile int& Mouse::x

Read-only coordinate of the mouse.

Note

Value is dependent on type of mouse used. Mice can return absolute coordinates or relative, where relative is internally accumulated.

const volatile int& Mouse::active

A read-only general timeout indicator to represent mouse activity in milliseconds.

Note

The value is clamped to 1000ms but is allowed to go above that for high activity while it is occurring.

Warning

Only when mouse polling is enabled through Mouse::startPolling.
